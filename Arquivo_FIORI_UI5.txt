3. Qual a diferença entre uma XML View e uma JS View? Com o XML view fica mais fácil de dizer quais os componentes UI, declarativa e todos os aplicativos Fiori são feitos com XML view. O JS View você define os componentes UI de forma imperativa e pode ser mais complexa pelo fato da lógica da UI estar misturada com o código do Controller.
